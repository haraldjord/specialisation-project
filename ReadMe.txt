Dette er filene som hører til BuoyancyVehicle Thesis - Embedded System for Underwater Data Acquisition

!! - BuoyancyFirmwareDocumentation - !!
Generert doxygen dokumentasjon av BuoyancyVehicle firmware:
	- BuoyancyFirmwareDocumentation - html - Åpne en av html.filene I webBrowser. (Testet med google Chrome: Version 83.0.4103.97 (Official Build)(64-bit) ).

!! - BuoyancyFirmwareFiles - !!
Buoyancy Vehicle firmware .c & .h filer - utgjør all skrevet og redigert programvare.
- WrittenFirmware: Enkeltstående filer
- BuoyancyFirmware: Prosjektmappe for Legger Embedded Studio - Eksempel på hvilke filer må/bør importeres fra Nordic SDK. Legges på samme mappenivå som Nordic SDK (SDK_dir).

!! - BuoyancyTerminalApp - !!
Buoyancy Terminal-applikasjon - InstallFiles inneholder filer før de er installert med nam. InstalledApplication inneholder filer etter fullført installasjon (med alle dependencies inkludert) i tillegg til mappen der overførte missionLog filer lagres. Den inneholder noen overførte filer - én mislykket, og to vellykkede.


!! - BuoyancyVehiclePlotTestData - !!
Matlab script for å lese og lage plot av BuoyancyVehicle missionLog data, og test av wireless luft-vann kommunikasjon
	

!! - MechanicalModificationStepFiles - !!
Inneholder step filer generert for maskinering og 3D-print av deler beskrevet i section: Mechanical Modifications
